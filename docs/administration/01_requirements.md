## Bootcamp Requirements

Bootcamp is a live session where mentors teach mentees various topics

- 4 meetings
- ~3 hours per meeting
- Mentor can take turns teaching
- Hands-on activities are mandatory
- Pre-release modules before sessions
- Post-tests after each module
- Evaluation metrics for mentees

## Module Requirements

Modules are structured learning materials provided to mentees

- Modules should include:
  - Objectives
  - Theoretical background
  - Practice questions with answer keys
  - Case studies
  - Installation guides for necessary software
