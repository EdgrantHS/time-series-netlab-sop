# Module 1: Data Exploration, Visualization, and Introduction to Python Notebooks

This directory contains the syllabus for Module 1, which covers the fundamentals of data exploration and visualization techniques essential for understanding time series data, along with an introduction to Python notebooks.

## Learning Objectives

By the end of this module, mentees should be able to:

- Set up and use Python notebooks (Jupyter)
- Perform basic data exploration
- Identify _trends_ and _seasonality_
- Calculate autocorrelation and partial autocorrelation
- Test for stationarity
- Create visualizations using Python libraries

## Topics Covered

This module is divided into the following topics:

1. **01_Python-Notebooks-Introduction.md** - Getting started with Jupyter notebooks
2. **02_Data-Exploration-Basics.md** - Fundamental data exploration techniques
3. **03_Trends-and-Seasonality.md** - Decomposing time series components
4. **04_Autocorrelation-Analysis.md** - Understanding autocorrelation patterns
5. **05_Stationarity-and-Differencing.md** - Testing and achieving stationarity
6. **06_Data-Visualization.md** - Creating time series visualizations

## Prerequisites

- Basic understanding of Python programming
- Familiarity with statistical concepts
