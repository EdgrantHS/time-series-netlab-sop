# Module 1: Machine Learning Fundamentals

This directory contains the syllabus for Module 1, which covers essential machine learning concepts, Python fundamentals, and data preprocessing techniques that form the foundation for time series analysis.

## Learning Objectives

By the end of this module, mentees should be able to:

- Set up and use Jupyter notebooks effectively
- Understand core AI and ML concepts
- Implement regression, classification, and clustering algorithms
- Build basic neural networks
- Apply best practices for data science workflows

## Topics Covered

This module is divided into the following topics:

1. **01_Jupyter-Notebooks-Setup.md** - Setting up and using Jupyter notebooks
2. **02_AI-ML-Background.md** - Introduction to artificial intelligence and machine learning
3. **03_Regression-Techniques.md** - Linear and non-linear regression methods
4. **04_Classification-Methods.md** - Classification algorithms and evaluation
5. **05_Clustering-Algorithms.md** - Unsupervised learning and clustering
6. **case_study** - This folder contains a case study that is taken from the "Machine Learning with Python" course on Cognitive Class. 

## Prerequisites

- Basic Python programming knowledge
- Understanding of mathematical concepts (algebra, statistics)
- Familiarity with data structures and algorithms
