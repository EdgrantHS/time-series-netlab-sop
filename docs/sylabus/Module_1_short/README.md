# Module 1: Machine Learning Fundamentals (2-Day Version)

This directory contains the condensed syllabus for Module 1, covering essential machine learning concepts and Python fundamentals in just 2 days.

## Learning Objectives

By the end of this module, mentees should be able to:

- Set up and use Python notebooks effectively
- Import and explore data using pandas
- Understand core ML concepts and problem types
- Implement basic regression and classification
- Perform essential data preprocessing

## Schedule Overview

**Day 1: Foundations and Data Handling**
- Topic 1: Notebooks and Data Basics (Full Day)
- Hands-on practice with Titanic dataset

**Day 2: Machine Learning Implementation**
- Topic 2: ML Fundamentals + Regression and Classification (Full Day)
- Hands-on practice with Boston Housing and Customer Churn datasets

## Topics Covered

This condensed module includes:

1. **01_Notebooks-and-Data-Basics.md** - Python notebooks + data importing/exploration + preprocessing essentials
2. **02_ML-Fundamentals-and-Implementation.md** - Core ML concepts + regression/classification implementation

## Prerequisites

- Basic Python programming knowledge
- Understanding of mathematical concepts (algebra, basic statistics)
